# 01.getting-started

本示例参考了官网 http://casperjs.org/ 的 demo ，并进行了简单的修改。

> CasperJS 的能力不仅局限于以下的示例中所展示，此处只是比较典型的应用场景举例。

## 1. NAVIGATION & AUTOMATION

CasperJS 自动导航和打开任意页面

```bash
casperjs sample.js
```

## 2. TESTING

CasperJS 支持测试

```bash
casperjs test sample-test.js
```

## 3. SCRAPING

CasperJS 支持抓取页面的元素

```bash
casperjs sample-scraping.js
```
